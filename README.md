# Инструкция по запуску проекта

Проект состоит из двух частей: Backend (NestJS + Prisma + PostgreSQL) и Frontend (Next.js). Для работы требуется установленный `Node.js` (рекомендуется v20+) и `Docker` (для базы данных).

---

## 1. Запуск Backend (Серверная часть)

Перейдите в папку бэкенда:
```bash
cd backend
```

1. **Установите зависимости:**
   ```bash
   npm install
   ```

2. **Запустите базу данных (PostgreSQL):**
   Убедитесь, что у вас запущен Docker, и выполните:
   ```bash
   docker-compose up -d
   ```

3. **Настройте переменные окружения:**
   Скопируйте пример файла конфигурации:
   ```bash
   cp .env.example .env
   ```
   *(По умолчанию в `.env` уже прописаны доступы к локальной БД, созданной через docker-compose)*

4. **Примените миграции базы данных:**
   Создаст таблицы в БД:
   ```bash
   npx prisma db push
   # ИЛИ
   npx prisma migrate dev
   ```

5. **Запустите сервер:**
   ```bash
   npm run start:dev
   ```
   Бэкенд будет запущен на `http://localhost:4000`.

---

## 2. Запуск Frontend (Клиентская часть)

Откройте **новый** терминал (не закрывая бэкенд) и перейдите в папку фронтенда:
```bash
cd frontend
```

1. **Установите зависимости:**
   Проект использует `bun` для быстрой установки, но вы можете использовать `npm`.
   ```bash
   bun install
   # ИЛИ
   npm install
   ```

2. **Настройте переменные окружения:**
   Создайте файл `.env` (если его нет) и добавьте туда ссылку на бэкенд:
   ```bash
   NEXT_PUBLIC_API_URL="http://localhost:4000"
   ```

3. **Запустите клиентское приложение:**
   ```bash
   bun dev
   # ИЛИ
   npm run dev
   ```
   Фронтенд будет запущен на `http://localhost:3000`.

---

Откройте [http://localhost:3000](http://localhost:3000) в браузере. Вы можете зарегистрировать новый аккаунт и сразу начать пользоваться приложением!
